# 接口：addScore  [返回](../README.md)
用例： [添加成绩](../用例/添加成绩.md)

- 功能：
    对学生的实验打分

- 权限：
    老师：必须先登录。

- API请求地址：
    接口基本地址/v1/api/addScore

- 请求方式 ：
    POST

- 请求实例：

        {
            "studentId":"201510414123",
            "name":"诗诗",
            "class":"一班",
            "a":"70",
            "b":"90",
            "c":"80"
        }

- 请求参数说明:

  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|
  |studentId|学生的id号，注意这不是学号|
  |name|姓名|
  |class|班级|
  |a| 完成度|
  |b|复杂度|
  |c|答辩成绩|

- 返回实例：

        {
            "status": true,
            "info": null,

        }

- 返回参数说明：

  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|
  |status|bool类型，true表示正确的返回，
